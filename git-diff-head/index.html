git diff HEAD

Posted on 2011-07-24 In git 
$ git diff HEAD

上面这条命令会显示你工作目录与上次提交时之间的所有差别，这条命令所显示的 内容都会在执行”git commit -a”命令时被提交。

这句等同于svn中的 svn diff。即比较工作副本与服务器之间的差异。
