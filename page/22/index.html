装修笔记——第13天，厨房贴墙砖

终于又开始新的工程了，铺砖，瓦工先从厨房的墙砖开始铺起，这个时候你还可以检验一下你家墙砖的质量，当墙砖被贴到墙上后，它的质量问题就很容易暴露出来了，如果上下的砖缝间隔距离不一样，有二种情况，第一种是您的瓦工铺砖水平不行，第二种情况，就是您的砖的质量不行，上下或左右砖缝间隔距离不一样，是上下或左右墙砖的大小（长度或宽度不一样）造成的。

检查墙砖的平整度，可以把两块砖的正面相对，放在一个平面上，查看四边的缝隙间距，如果太大或间距不一至，说明砖面是拱型的，平整度存在质量问题。

装修笔记——第10天，墙砖已换

墙砖已经换了，但还是没有想象中的理想！不可能把每个砖都看得那么仔细。

总结的经验是

  1. 为了避免类这样的问题，可以考虑到东方家园之样的建材超市去买，消费者面对的是超市，不是个人，类似问题相对好解决（遇到过由于不同批次造成瓷砖色差问题给退/换货的，其实不同批次的瓷砖有色差是正常现象，如果是面对个人经营的话恐怕很难给退/换货了）
  2. 看墙砖的时候一定要仔细看好样砖，我这个就属于当时没有仔细看，结果今天找过去的时候再次看购买的墙砖，发现样砖上也有黑点。
  3. 一分价钱一分货，绝对的！虽然我买的墙砖上面有小黑点，最多就是对美观造成那么一点点的小影响，但砖的质量还是相当好的。

装修笔记——第8天，改水打压测试

在改完水电后，要做的工作就是对所有改水的地方进行打压测试，正常的压力测试是0.8pa，我家做了1个小时，看网上说有做一天的，做打压测试的目的是检查水路有无漏水，水管接口出是否紧密，PPR管是否有开裂的情况，打压试验结束时，如果指针还指在0.8pa处，说明水路没有问题，如果小于0.8pa，则水路改造有问题，这时就要查找问题出在以上环节的哪个地方了。

ubuntu 字幕乱码

ubuntu 使用Totem Movie Player播放时，字幕乱码，解决方法如下

方法一
点击 edit -> preferences ：subtitles下的 font选项，选择Chinese Simplified 。重新载入字幕即可。

Totem Movie Player默认播放时是不自动加载字幕的，可以在 “automatically load subtitle file when movies loaded”前面打勾，这样就可以在播放影片时自动加载字幕了。

方法二
由于字幕的编码多是gbk，所以在ubuntu下把文件编码转换为utf8，字幕就可以正常显示了。

iconv -f gbk -t utf8 [暮光之城].Twilight.DVDRip.XviD-DiAMOND-cd1.srt >[暮光之城].Twilight.DVDRip.XviD-DiAMOND-cd1-utf8.srt

PS：通过以上方法虽然能够解决字幕乱码问题，但是，如果你拉动进度条，字幕将消失，目前这个问题还没有解决。
