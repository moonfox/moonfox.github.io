wc命令

Posted on 2011-09-03 In 操作系统

wc命令用来打印文件的文本行数、单词数、字节数等（print the number of newlines, words, and bytes in files）

格式：wc -l <file>

打印指定文件的文本行数。（l=小写L）

参数：-c, --bytes
打印字节数（print the byte counts）

参数：-m, --chars
打印字符数（print the character counts）

参数：-l, --lines
打印行数（print the newline counts）

参数：-w, --words
打印单词数（print the word counts）

memcached 启动参数

Posted on 2011-09-02 In 后端

启动参数

-p memcached监听的TCP端口
-l 监听的ip地址
-d 以daemon方式运行，将程序放入后台
-u memcached的运行用户
-P memcached的pid文件路径
-m memcached可以使用的最大内存数量
-c memcached同时可以接受的最大的连接数
