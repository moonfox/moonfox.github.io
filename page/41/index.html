Belongs_to选项中的primary_key

Posted on 2009-10-08 In 程序开发 Symbols count in article: 366 Reading time ≈ 1 mins.

在定义mode时给belongs_to加入了:primary_key选项，运行时报错，提示Unknown primary-key。去网上寻找答案，在rubyonrails的官方网站的关于rails2.3.3新特性介绍中看到有这么一段描述

:primary_key option for belongs_to for broader support of legacy schemas and those using a separate UUID primary key: belongs_to :employee, :primary_key => ‘SSN’, :foreign_key => ‘EMPID’

原来直到rails2.3.3时primary_key才被belongs_to支持，偶用的是rails2.3.2，难怪会出现Unknown primary-key错误。

Ubuntu Gedit 乱码

Posted on 2009-10-08 In 我爱Linux Symbols count in article: 235 Reading time ≈ 1 mins.

在ubuntu中用gedit打开gb2312的文件时，会出现乱码。

解决方法a

  1. 进入终端
  2. shell>gconf-editor
  3. 依次点击apps->gedit-2->preference->encodings
  4. 在auto_detected和show_in_menu两个list类型的键中加入GBK或GB18030（注意：要把这两个值调到列表最前面）

解决方法b
在终端录入 gedit –encoding=编码 filename也可以解决打开gb2312文件乱码问题

地铁4号线 通车

Posted on 2009-09-28 In 让资讯飞 Symbols count in article: 322 Reading time ≈ 1 mins.

举世瞩目的4号线，经过两年紧张的建设，于今日上午10时，正式开通了。其北起安河桥北，南到公益西桥，全长28.2公里。

晚上回家的时候去体验了一把，从西四上的车。坐到西单，然后换乘1号线。总的感觉是
1、从西四进站的时候，扶梯只有上的，没有下的，要进站的话只能走楼梯，而且感觉坡度还比较徒，恐怕稍微上点年纪的人都会吃不消的。

2、进站距离有点远，一路左拐右拐才到检票口

3、全程手机有信号（唯一值得表扬的地儿）

4、西单换乘的路线太远，又是扶梯又是长廊，然后又一通左拐右拐。
总结：主要是换乘路线太远（指西单站，别的站没坐过，不清楚），从西四坐14路到和平门地铁坐2号线，然后再换乘到复兴门的时间，要比坐4号线快得多。决定以后还是继续坐14路，再换乘2号线。

大象腿、游泳圈、拜拜袖

Posted on 2009-09-26 In 杂七杂八 Symbols count in article: 401 Reading time ≈ 1 mins.

其实现在发胖的人并不是很多，但为什么那多的人看上去还是那么的“胖呢”，其实就是他们身上有“大象腿、游泳圈、拜拜袖”这几样东西造成的。

“大象腿”

指的就是按身材比例说，腿比较偏胖的人，这里主要指的是女性。同样的是长脂肪，女性容易向下长，表现出来的就是腿胖，男性容易向上长，表现出来的就是大肚子。
减脂方法：全民体质检测运动 ：童子拜佛。

“游泳圈”

指的就是腰上的赘肉，有的人身上不指一个”游泳圈“，严重者站着的时候有二个，坐下的时候有三个。
减脂方法：平时可以注意一下坐姿，不要半躺，更不能全躺在座椅里，坐的时候身体要向前挺直一些，个人感觉这个姿势不但有助于减脂，还有助于提起精神，提高工作效率。最好能在空闲的时候做一些”板凳操“之类的运动。

“拜拜袖”

就是指大臂上的赘肉，如果您不明白的话接着向后看，就是有”拜拜袖”的人挥手向你说再见的时候，他的手还没动，胳膊上半支的那块肉就已经不停的在前后摇摆了。

减脂方法：待续
