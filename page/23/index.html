ubuntu virtualbox 网银

由于操作系统的限制，ubuntu里不能使用网银，要么安装双系统（这种方法成本太高，因为要白白浪费一个分区），要么装虚拟机，难点在于如何让虚拟机识别网银，即如何识别USB设备，是一大难题。

这里说一下ubuntu10.04 + virtualbox3.2.6 使用网银的方法。

本人的解决方法是。
1、在virtualusers组中添加当前登录用户
sudo adduser 当前登录用户 virtualusers
2、重新启动电脑，重新登出再登入也可以。
3、在virtualbox中启用usb设备，点击设置->USB设备->启用。如果没有此选项，请到官方网站重新下载个最新版装上就有了，通过新立德安装的一般都没有usb选项。

ubuntu 自动加载ntfs分区

自动挂载的目的在于可以方便的访问windows 下的分区，例如把C盘挂载到/mnt/windows/c下，这样直接录入路径就可以访问到C盘了，虽然ubuntu可以通过开机后手动点击加载分区，系统会为点击的分区自动分配一个十分难看的字符串去标识，本博对这一点就十分的讨厌，而且每次点击后分配的标识都不一样， 这样对于类似virtualbox这样的软件在读取光盘镜像这样的文件时，会因标识符的变化而无法加载。

如何加载：
1、首先要建立挂载的目标文件夹，也就是你要把分区挂载到哪个文件夹下面。本博全部分别挂载到/mnt/windows/c、d、e、f下面。如果你也要挂载到/mnt下面，请自行新建文件夹
2、记录分区UUID值。每个分区都有UUID值。使用ls -l /dev/disk/by-uuid进行查看，会得到类似“3458F2ABXXXXXX4E -> ../../sda1” 这样的值，很明显，sda1对应的就是C盘，其UUID值是3458F2ABXXXXXX4E。
3、编辑/etc/fstab文件，在最后一行加入UUID=3458F2ABXXXXXX4E /mnt/windows/c ntfs-3g uid=1000,gid=0,umask=133,dmask=022,utf8 0 0。这样在开机的时候，系统会自动把C分区挂载到 /mnt/windows/c目录下。
4、重新启动电脑，在地址栏里直接录入 /mnt/windows/c就应该可以看到C盘的内容了。

ubuntu configure: error: Berkeley DB not found

为了让svn 支持Berkeley DB，在编译安装时必须在安装apr-util时加入 –with-berkeley-db，将 Berkeley DB编译进去，在configure时出现configure: error: Berkeley DB not found错误，原因是系统没有找到libdb-yourversion.so文件，如果你已经安装了Berkeley DB ，在安装目录的lib库里找到此文件，链接到/usr/lib下即可。

以本人的机子为例
cd /usr/local/lib
ln -s /opt/bdb/lib/libdb-4.7.so
ldconfig

在重新配置一下就好了。

装修——如何选择居室门

分类：

1、模压门:
是用一种具有各种凹凸图案及光面的高密度纤维板做门面的门，门面采用木材纤维一次模压成型。
此类居室门在加工时，胶水应达到环保要求，否则会影响健康。模压门是工厂一次模压成型的，价格便宜，质感差，隔音效果不好，款式比较单一，属中低档产品。模压门只适宜做混油门，尤其是白色混油。
2、实木门:
实木门的说法有两种:由内至外全部为实木或芯材为多层板制作的都叫实木门。实木门采用传统工艺加工而成，以烘干的松木、铁杉木、楸木、水曲柳等为基材，其结构主要为门框榫连接，镶嵌门芯板，在市场上属于中高档产品。全实木门在北方的气候条件下使用存在一定的缺陷，主要表现为门扇变形，榫连接处风干开裂，门芯板收缩露白。
