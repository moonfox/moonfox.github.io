SSD关键优势

SSD在过去几年里一直在迅速发展，现在似乎已经到了一个阶段，在某些应用上它们可以胜过HDD。SSD提供的关键优势包括：

用电量更低——服务器上一块SSD的功耗大概是3瓦，而类似的一块HDD则在6至10瓦之间。
更快的查寻速度——如上所述，SSD的查寻时间在随机读取方面至少比HDD要快得多：SSD的查寻时间约为100微秒，而HDD平均是4500微秒。
更高的可靠性——缺少活动部件，意味着SSD出现硬盘硬件故障的可能性比HDD要小得多。根据公布的数据，SSD的平均无故障时间（MTBF）约为200万小时，而HDD则仅为其六分之一，约为30万小时。
工作静音——SSD运行时完全静音，硬盘驱动器则绝对不可能。

什么是 Client Access License （CAL）

什么是 Client Access License （CAL）？

如果您公司的工作站是互联的，您将会使用一台网络服务器，其它客户端来访问服务器端软件来实现某些例如文件和打印共享的功能。 Client Access License 或者叫 CAL 用来为服务器端软件提供合法访问授权。 CAL 并不是一个软件产品，它是一个赋予用户访问服务器端服务权限的授权。

安装dabr必要条件

在你安装dabr时，请确认已满足以下必要条件

  1. php版本 5.2＋因为要用到 json_decode() function)
  2. 确保主机支持 mcrypt 模块
  3. 确保主机支持 mod_rewrite 模块

偶在安装后运行时遇到

  1. Parse error: syntax error, unexpected '=', expecting ')' in /data/home/zz43d1/htdocs/dabr/common/user.php on line 52
    可能就是php版本低于5.2造成的，偶用的是万网的主机，为php4.4.9。
  2. 如果主机不支持 mcrypt 模块，可以用第三方方法解决。这个在网上已经有方案了。

ubuntu ssh 翻墙

如何在 ubuntu下通过ssh翻墙呢
1、你有一台国外主机
2、你的主机支持SSH
3、你的主机商不介意你使用 SSH 隧道功能

具备以上条件后，你就可以在ubuntu 下 用 SSH进行翻墙了
1、ssh -qTfnN -D 端口 user_name@host_home
2、Firefox+AutoProxy扩展，指定你的代理为127.0.0.1 端口为 刚才设定的那个端口就可以了。

行了，现在你已经自由了。
