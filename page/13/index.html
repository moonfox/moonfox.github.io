ajax尽量使用get

When using the XMLHttpRequest object, the browser implements POST in two steps: (1) send the headers, and (2) send the data. It is better to use GET instead of POST since GET sends the headers and the data together (unless there are many cookies). IE’s maximum URL length is 2 KB, so if you are sending more than this amount of data you may not be able to use GET.

Post先发送head，再发送data
Get是 head/data一起发送

佳能镜头扫盲- 红圈.大三元.小三元.狗定.牛定.小痰盂(转载)

对一些初次接触佳能的用户，在挑选镜头的时候经常看到一些奇怪的名称，一头雾水，不知所云。这里根据自己的经验和认识，简单说明一下。如果有误，请指正：

红圈

镜头前有一红色的圈圈为标志。按照佳能的定义，“红圈”就是镜头型号里面带有“L”字的，Luxury - 高级豪华的意思

50mm f1.2 L - 就是指50毫米，定焦，恒定光圈1.2，L 红圈镜头；
70-200mm f2.8 IS L - 就是指　70-200毫米，变焦，恒定光圈２.8, 带ＩＳ防震，Ｌ红圈镜头。

“红圈”头一般都是恒定光圈的（几个超广角，或超长焦的镜头除外），有一定的防水防尘功能，做工精良，采用最高级的光学镜头材料，成像素质极佳。
