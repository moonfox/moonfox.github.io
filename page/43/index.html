苹果键盘 开启大写 字符不正确

Posted on 2009-09-24 In 日志

登录网上银行，录密码时居然三次全被提示错误。难道记错了。第二天又接着登录，还是三次全错。开始怀疑自己的记性有问题。第三天，又是三次全错，彻底崩溃，已经相信密码被盗且修改之（由于有U盾，所以不是十分担心），已经做好明天去银行重置密码的准备。

突然怀疑键盘有问题，于是将密码打在记事本上~~~果然，当开启caps lock时，字母居然显示的不正确。只好改用shift键，成功登录。其实开启caps lock（大写状态）是正确的做法，按住shift是非正确的做法，而这次错误的原因，就是由于使用了正确的做法，ORZ一下自己。想起前几天支付宝也偶到了类似情况，可能也是这个原因造成的。叵测的苹果键盘，虽然给偶带来了这么多的麻烦，但依然爱你！

Firefox 收藏夹 导入 IE

Posted on 2009-09-19 In 软件

莫莫同学打来电话，说firefox有些问题，问能不能把收藏夹里的书签导入IE。这还真给偶问住了，以前从没弄过啊（基本不用IE），上网搜索，解决方法如下

1、打开firefox，点击书签－管理书签－导入以及备份－导出HTML－保存（默认文件名为bookmarks.html）
2、用记事本打开bookmarks.html，点击 另存为－选择编码方式为ANSI，保存文件
3、打开IE，点击 文件－导入与导出－下一步－导入收藏夹－选择bookmarks.html －下一步－选中收藏夹－完成。

Posted on 2009-09-16 In 数据库

在SQL Server数据库中事务日志起着至关重要的作用：

它持续记录数据库所有的事务和这些事务对数据库所做的修改;一旦数据库出现灾难事件，就需要事务日志来进行近期数据的恢复操作。

Ubuntu 升级Firefox 通过Apt源列表

Posted on 2009-09-15 In 日志

升级firefox的一般的安装方式是去firefox的官方网站下载tar.bz2包，解压后直接运行即可。另一种安装方式是通过apt-get 方式安装。

第一步、添加下列源
deb http://ppa.launchpad.net/ubuntu-mozilla-security/ppa/ubuntu jaunty main
deb-src http://ppa.launchpad.net/ubuntu-mozilla-security/ppa/ubuntu jaunty main

第二步、获取key
sudo apt-key adv –keyserver keyserver.ubuntu.com –recv-keys 12345678

第三步、更新源
sudo apt-get update

第四步、安装

Ubuntu Flash 中文 乱码

Posted on 2009-09-13 In 操作系统

由于ubuntu9.04是英文环境，导致在flash中的文本字体全是乱码，请注意，这里所指的乱码不是一个个方框的那种，而是一些符号。网上的解决办法大都是删除/etc/fonts/conf.d/49-sansserif.conf文件、要么就是把此文件的倒数第4行的字体设置为系统所支持的中文字体，这种办法只对ubuntu是中文环境的情况下起作用，在英文环境下依旧不能解决问题。如果有谁解决了，麻烦请告之方法。在这里先谢谢了！

升级Opera及字体设置

Posted on 2009-09-12 In 日志

在升级opera时，请先卸载掉以前的旧版本，不要直接升级或重新安装，应将其完全卸载后，再进行安装。否则会出现lock锁定提示，安装将失败！安装好后，最好进行一下字体的设置，大多数情况下，opera默认的字体非常难看，在这里建议使用微软雅黑的字体（纯属个人意见），比较漂亮。
