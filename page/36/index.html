ubuntu的下载加速器

在ubuntu中，当你通过源更新系统时，如果此源比较慢的话，可以考虑使用axel ，它可以进行分段下载，从而加快你的下载速度。本人在更新某个国外源软件时下载速度非常慢，大概10K左右，在使用axel后，下载速度为56K（本人ADSL 512），效果还是相当明显滴！

sudo apt-get –print-uris -y dist-upgrade | grep http | awk ‘{print $1}’ | sed “s/‘//g” | xargs -l1 sudo axel -n5 -a -o /var/cache/apt/archives/

地铁4号线 通车

举世瞩目的4号线，经过两年紧张的建设，于今日上午10时，正式开通了。其北起安河桥北，南到公益西桥，全长28.2公里。

晚上回家的时候去体验了一把，从西四上的车。坐到西单，然后换乘1号线。总的感觉是
1、从西四进站的时候，扶梯只有上的，没有下的，要进站的话只能走楼梯，而且感觉坡度还比较徒，恐怕稍微上点年纪的人都会吃不消的。
2、进站距离有点远，一路左拐右拐才到检票口
3、全程手机有信号（唯一值得表扬的地儿）
4、西单换乘的路线太远，又是扶梯又是长廊，然后又一通左拐右拐。
总结：主要是换乘路线太远（指西单站，别的站没坐过，不清楚），从西四坐14路到和平门地铁坐2号线，然后再换乘到复兴门的时间，要比坐4号线快得多。决定以后还是继续坐14路，再换乘2号线。

大象腿、游泳圈、拜拜袖

其实现在发胖的人并不是很多，但为什么那多的人看上去还是那么的“胖呢”，其实就是他们身上有“大象腿、游泳圈、拜拜袖”这几样东西造成的。

“大象腿”指的就是按身材比例说，腿比较偏胖的人，这里主要指的是女性。同样的是长脂肪，女性容易向下长，表现出来的就是腿胖，男性容易向上长，表现出来的就是大肚子。
减脂方法：全民体质检测运动 ：童子拜佛。

“游泳圈” 指的就是腰上的赘肉，有的人身上不指一个”游泳圈“，严重者站着的时候有二个，坐下的时候有三个。
减脂方法：平时可以注意一下坐姿，不要半躺，更不能全躺在座椅里，坐的时候身体要向前挺直一些，个人感觉这个姿势不但有助于减脂，还有助于提起精神，提高工作效率。最好能在空闲的时候做一些”板凳操“之类的运动。

“拜拜袖” 就是指大臂上的赘肉，如果您不明白的话接着向后看，就是有”拜拜袖”的人挥手向你说再见的时候，他的手还没动，胳膊上半支的那块肉就已经不停的在前后摇摆了。
减脂方法：待续

苹果键盘 开启大写 字符不正确

登录网上银行，录密码时居然三次全被提示错误。难道记错了。第二天又接着登录，还是三次全错。开始怀疑自己的记性有问题。第三天，又是三次全错，彻底崩溃，已经相信密码被盗且修改之（由于有U盾，所以不是十分担心），已经做好明天去银行重置密码的准备。

突然怀疑键盘有问题，于是将密码打在记事本上~~~果然，当开启caps lock时，字母居然显示的不正确。只好改用shift键，成功登录。其实开启caps lock（大写状态）是正确的做法，按住shift是非正确的做法，而这次错误的原因，就是由于使用了正确的做法，ORZ一下自己。想起前几天支付宝也偶到了类似情况，可能也是这个原因造成的。叵测的苹果键盘，虽然给偶带来了这么多的麻烦，但依然爱你！
