Ubuntu 字幕乱码

Posted on 2010-08-07 In 我爱Linux

ubuntu 使用Totem Movie Player播放时，字幕乱码，解决方法如下

方法一
点击 edit -> preferences ：subtitles下的 font选项，选择Chinese Simplified 。重新载入字幕即可。

Totem Movie Player默认播放时是不自动加载字幕的，可以在 “automatically load subtitle file when movies loaded”前面打勾，这样就可以在播放影片时自动加载字幕了。

方法二
由于字幕的编码多是gbk，所以在ubuntu下把文件编码转换为utf8，字幕就可以正常显示了。

iconv -f gbk -t utf8 [暮光之城].Twilight.DVDRip.XviD-DiAMOND-cd1.srt >[暮光之城].Twilight.DVDRip.XviD-DiAMOND-cd1-utf8.srt

PS：通过以上方法虽然能够解决字幕乱码问题，但是，如果你拉动进度条，字幕将消失，目前这个问题还没有解决。

Ubuntu 自动加载Ntfs分区

Posted on 2010-08-03 In 我爱Linux

自动挂载的目的在于可以方便的访问windows 下的分区，例如把C盘挂载到/mnt/windows/c下，这样直接录入路径就可以访问到C盘了，虽然ubuntu可以通过开机后手动点击加载分区，系统会为点击的分区自动分配一个十分难看的字符串去标识，本博对这一点就十分的讨厌，而且每次点击后分配的标识都不一样， 这样对于类似virtualbox这样的软件在读取光盘镜像这样的文件时，会因标识符的变化而无法加载。

如何加载：

  1. 首先要建立挂载的目标文件夹，也就是你要把分区挂载到哪个文件夹下面。本博全部分别挂载到/mnt/windows/c、d、e、f下面。如果你也要挂载到/mnt下面，请自行新建文件夹
  2. 记录分区UUID值。每个分区都有UUID值。使用ls -l /dev/disk/by-uuid进行查看，会得到类似3458F2ABXXXXXX4E -> ../../sda1 这样的值，很明显，sda1对应的就是C盘，其UUID值是3458F2ABXXXXXX4E。
  3. 编辑/etc/fstab文件，在最后一行加入UUID=3458F2ABXXXXXX4E /mnt/windows/c ntfs-3g uid=1000,gid=0,umask=133,dmask=022,utf8 0 0。这样在开机的时候，系统会自动把C分区挂载到 /mnt/windows/c目录下。
  4. 重新启动电脑，在地址栏里直接录入 /mnt/windows/c就应该可以看到C盘的内容了。

装修——如何选择居室门

Posted on 2010-07-25 In 杂七杂八

分类：

  1. 模压门:

    是用一种具有各种凹凸图案及光面的高密度纤维板做门面的门，门面采用木材纤维一次模压成型。
    此类居室门在加工时，胶水应达到环保要求，否则会影响健康。模压门是工厂一次模压成型的，价格便宜，质感差，隔音效果不好，款式比较单一，属中低档产品。模压门只适宜做混油门，尤其是白色混油。

  2. 实木门:

    实木门的说法有两种:由内至外全部为实木或芯材为多层板制作的都叫实木门。实木门采用传统工艺加工而成，以烘干的松木、铁杉木、楸木、水曲柳等为基材，其结构主要为门框榫连接，镶嵌门芯板，在市场上属于中高档产品。全实木门在北方的气候条件下使用存在一定的缺陷，主要表现为门扇变形，榫连接处风干开裂，门芯板收缩露白。

网站301重定向

Posted on 2010-07-24 In 我爱Linux

gagahappy.com这个域名已经用了一段时间了，想把gagahappy.cn自动重定向到gagahappy.com上面，本来打算用万网的服务去做（收费要200元），想到自己的主机是apache的，可以用其自身的重定向功能。

由于用的是虚拟主机，修改.htaccess文件即可。

RewriteCond %{HTTP_HOST} ^www.gagahappy.cn [NC]
RewriteRule ^(.*)$ http://www.gagahappy.com/$1 [L,R=301]

装修——外墙和顶棚的抹灰层

Posted on 2010-07-23 In 杂七杂八

经调研发现混凝土（包括预制混凝土）顶棚基体抹灰，由于各种因素的影响，抹灰层脱落的质量事故时有发生，严重危及人身安全，引起了有关部门的重视，如北京市为解决混凝土顶棚基体表面抹灰层脱落的质量问题，要求各建筑施工单位，不得在混凝土顶棚基体表面抹灰，用腻子找平即可，5年来取得了良好的效果。
